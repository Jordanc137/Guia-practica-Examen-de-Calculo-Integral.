# Jordan Academy | Cálculo Integral Avanzado

Este repositorio contiene una serie de guías estratégicas y resoluciones detalladas de cálculo integral, desarrolladas íntegramente en LaTeX. El objetivo de este proyecto es transformar el aprendizaje de las matemáticas superiores a través del rigor metodológico y la claridad visual.

## Filosofía del Proyecto

En **Jordan Academy**, entendemos que la resolución de problemas complejos no depende de la memorización, sino del reconocimiento de patrones y la estructuración del pensamiento. Estas guías no solo presentan soluciones, sino el **algoritmo de decisión** detrás de cada método.

## Contenido

* **Estrategia de Integración:** Manual para la identificación de métodos (Sustitución, Partes, Trigonométrica, Fracciones Parciales).
* **Solucionario Detallado:** Ejercicios de alta complejidad resueltos paso a paso con un enfoque pedagógico.
* **Plantillas LaTeX:** Código fuente diseñado con una estética académica profesional (utilizando paquetes como `tcolorbox`).

## Cómo utilizar este material

1.  **PDFs:** Consulta las resoluciones finales para estudiar la metodología.
2.  **Código Fuente (.tex):** Puedes utilizar los archivos fuente como base para tus propios documentos académicos o para personalizar el estilo visual.
